## Proyectos destacados

### [`clinicode-api`](https://github.com/JaderFlorez/clinicode-api)
> API REST backend para un sistema clínico — **Node.js · TypeScript · Fastify · Supabase · PostgreSQL · Zod · Jest**

API completa para gestión de pacientes, médicos, consultorios y citas médicas. Implementada con **arquitectura hexagonal** real: dominio, aplicación, infraestructura y presentación completamente desacoplados. Incluye validación con Zod, pruebas unitarias e integración con Jest + Supertest, colección de endpoints en Bruno, migraciones SQL con UUIDs y ENUMs, y validación de traslapes de horarios en asignación médico-consultorio.

---

### [`calculadora_nomina_con_tkinter`](https://github.com/JaderFlorez/calculadora_nomina_con_tkinter)
> Calculadora de nómina colombiana — **Python · Tkinter · PyInstaller**

Aplicación de escritorio con lógica de negocio real: cálculo de horas extras diurnas, nocturnas, dominicales y festivas, aportes a salud y pensión, auxilio de transporte y días de vacaciones. Implementada con POO, múltiples ventanas y validaciones de entrada. Empaquetada como ejecutable `.exe` con PyInstaller.
